# 민주적 점심 🍚 — 교재 프롬프트로 만든 점심 투표 앱

서울대 평생교육원 「데이터 기반 바이브 코딩」 4일차 교재의 **① 샘플 프롬프트와 ③ 프롬프트 개선**을
그대로 AI에게 넣어 받은 코드를, 단계별 페이지로 묶은 저장소입니다.

- 교재: https://greatsong.github.io/snu-data-2026/day4.html (2절 '민주적 점심')
- 이 코드는 교재의 ② 예시 블록과 글자 단위로 동일하며, 실행 검증(AppTest 9개 시나리오)과
  프롬프트 강건성 실측(2모델 × 5회 반복 생성)을 거쳤습니다.

## 페이지 구성

| 페이지 | 무엇 | 교재 대응 |
|--------|------|-----------|
| 1단계 · 기본판 | 이름·메뉴 투표 → 1위 발표 → 잔소리 → '먹었다' 기록 → 표 2개 | ① 샘플 프롬프트 |
| 2단계 · 개선판 | 개표 방송 말투 · 풍선 · 가로 막대그래프 · 주간 그래프 · 동점 결선 | ③ 프롬프트 개선 |

두 페이지는 같은 시트를 씁니다. 1단계에서 넣은 표가 2단계에도 그대로 보여요.

## 준비 — 구글 시트 + 접수창구 (5분)

1. [sheets.new](https://sheets.new)로 새 시트 → 1행에 **시각 · 팀원 · 메뉴 · 구분** 네 칸
2. 확장 프로그램 → Apps Script → 기존 내용 지우고 [`apps_script/Code.gs`](apps_script/Code.gs) 붙여넣고 저장
3. 배포 → 새 배포 → 유형: **웹 앱** → 실행: **나** · 액세스: **모든 사용자** → 배포 → 권한 승인
4. `/exec`로 끝나는 **웹 앱 URL** 복사

자세한 따라하기(중간 점검 포함)는 교재 2-1절에 있습니다.

## 실행

`.streamlit/secrets.toml`(로컬) 또는 스트림릿 클라우드 Settings → Secrets:

```toml
SHEET_URL = "https://script.google.com/macros/s/…/exec"
```

```bash
pip install -r requirements.txt
streamlit run app.py
```

스트림릿 클라우드 배포 시 Main file path를 `app.py`로 지정하세요.
